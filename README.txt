Lenguaje: Python 3.9.0 64-bit
Visual Studio Code | versión 1.51.1
Software de terceros: pygame, OpenGl.

Manual de usuario: 

Se utiliza únicamente el teclado para jugar.
El personaje se mueve hacia adelante y hacia atrás con las teclas 'w' y 's' respectivamente.

Se puede rotar la cámara de manera horizontal con las teclas 'a' y 'd' (izquierda y derecha), también se puede 
realizar las mismas acciones con las teclas de las flechas ('LEFT' y 'RIGHT'). Si se usan en conjunto se duplica 
la velocidad de rotación o se cancelan entre sí.

Con las flechas de arriba y abajo (teclas 'UP' y 'DOWN') se puede rotar la cámara de manera vetical en torno al 
personaje.

Con la barra espaciadora (tecla 'SPACE') se realiza la acción de saltar.

Con las teclas 'e' y 'q' se realiza un zoom-in o zoom-out respectivamente.

Con la tecla 'k' se realiza un movimiento de ataque.
Con la tecla 'j' se realiza un gesto de celebración.


Con la tecla 'l' se activa y desactiva la iluminación de la escena que está configurada como un modo nocturno.

Con la tecla 'm' se pueden ver los polígonos dibujados.
Con la tecla 'z' se activa y desactiva el Z-BUFFER.
Con la tecla 'b' se activa y desactiva el backface culling.
Con la tecla 'c' se cambia la orientación de los polígonos al frente.
